### gdb常用命令
- gdb + 可执行文件 --> 进入gdb调试模式
- run -->直接执行程序
- start --> 启动，停留在main函数，分步调试
- n(ext) --> 下一条指令
- s(tep) --> 下一条指令，可以进入函数内部，但是库函数进不去
- q(uit) --> 退出gdb
- set args ... --> 设置启动参数
- b(reak) (行号 | 函数名 |源文件名:行号) --> 设置断点
- l(ist) (空 |源文件:行号) --> 查看源码(默认显示10行)
- d(el) (break number) --> 删除断点
- i(nfo) b --> 查看所有断点
- c(ontinue) --> 跳转到下一个断点
- p(rint) 变量名 --> 打印变量值
- ptype 变量名 --> 打印变量的类型
- set argc = 4 --> 可以给变量赋值
- display 变量名 --> 跟踪变量的值变化
